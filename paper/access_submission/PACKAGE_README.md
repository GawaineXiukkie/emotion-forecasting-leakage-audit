## 投稿前最后确认

- 没有任何 ICASSP 或其他版本同时处于审稿状态；
- 如果是已发表会议论文的扩展版，正文已引用会议版，cover letter 已说明新增内容；
- 作者姓名、单位、邮箱、biography、ORCID、资助和利益冲突准确；
- Acknowledgment 中的 AI 使用披露准确；
- IEEE 系统转换出的 proof 与本包 PDF 一致；
- 系统关键词填写 3--10 个，建议沿用论文中的 5 个关键词；
- submitting author 的 ORCID 公开且资料完整。
